### Project Overview
- **Objective**: Develop a dashboard for merchants that includes an upload menu button, lists all merchants with their last upload date, and displays daily and monthly stats.
- **Core Features**:
  - Merchant List
  - Upload Menu Button
  - Stats Display
  - File Handling

### Feature Implementation

#### Merchant List
- **Description**: Display all merchants with their last upload date and daily/monthly stats.
- **Development Notes**:
  - API endpoint: `/api/merchants`
  - Frontend component: `MerchantList`

#### Upload Menu Button
- **Description**: Allow merchants to upload CSV files and update the database.
- **Development Notes**:
  - API endpoint: `/api/upload`
  - Frontend component: `UploadButton`
  - CSV handling libraries: `csv-parser`, `fast-csv`

#### Stats Display
- **Description**: Calculate and display daily and monthly stats for each merchant.
- **Development Notes**:
  - API endpoint: `/api/stats`
  - Frontend component: `StatsDisplay`

#### File Handling
- **Description**: Handle two input file types: Merchant Demographic file and Transaction file. Output combined transaction CSV and merchant demographics exports.
- **Development Notes**:
  - Input files: `merchant_demographics.csv`, `transactions.csv`
  - Output files: `combined_transactions.csv`, `merchant_demographics_export.csv`
  - CSV handling libraries: `csv-parser`, `fast-csv`

### Technical Requirements

- **Backend**: Node.js with Express
- **Database**: MongoDB or PostgreSQL (deployable to AWS and other cloud platforms)
- **CSV Handling**: `csv-parser`, `fast-csv`
